# Places App Frontend

The frontend is a Next.js based React application that provides a fast, modern UI for exploring places and related data. It is built with TypeScript and follows a component-driven architecture for scalability and maintainability.

**Tech Stack:**
Next.js – App framework (SSR, routing, API integration)
React – Core UI library
TypeScript – Type safety and better developer experience
TanStack Query – Data fetching, caching, and synchronization
Tailwind CSS – Styling system
shadcn/ui - UI components

## Requirements

- Node.js 20+
- pnpm 9

Open the frontend at `http://localhost:3000`.

## Run Locally

From the repository root, install dependencies:

```bash
pnpm install
```

Start the frontend in development mode:

```bash
pnpm dev
```

Build:

```bash
pnpm build
```

Start the production build:

```bash
pnpm start
```
